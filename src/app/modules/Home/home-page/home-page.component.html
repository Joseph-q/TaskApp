Aplicación de Tareas

Este proyecto es una aplicación de tareas desarrollada en Angular 19, que se comunica a través de una Web API hecha en .NET.

  • Es necesario registrarse y loguearse para utilizar la aplicación.
  • Una vez logueado, puedes:
    • Ver tus tareas
    • Agregar nuevas tareas
    • Editar tareas existentes
    • Eliminar tareas
